# vue3-koa2-stage

---
## 一、项目介绍
### 前端

- `vue3 + typescript`
- `vue-router@next - history mode`
- `elementui-plus`

### 后端

- `koa2`
- 基于装饰器的控制层
- 函数式编程 - Ramda
## 二、项目使用
### 依赖安装

- npm install

### 本地开发

- `npm run dev-s`
- `npm run serve`
- 访问 `localhost:8080`

#### 项目提交

- 【前置】全局安装 `commitizen`
- 【前置】选择适配器 `echo '{ "path": "cz-conventional-changelog" }' > ~/.czrc`
- `npm run commit`
### 生产环境

- `npm run build`
- `npm run release`
- 访问 `localhost:3000/stage`
